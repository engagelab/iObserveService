Add a new resource

POST /resource

Request

{
	"type": "brochure"
}

Parameters

  • type <string>

Response

{
  "_id" : "516fe13f0ac19c3ed7000001",
  "type" : "brochure"
}

 

 

List all resources

GET /resource

Response

[
  {
    "_id" : "516fe13f0ac19c3ed7000001",
    "type" : "brochure"
  }
]

 

 

Get a resource by resource id

GET /resource/516fe13f0ac19c3ed7000001

Response

{
  "_id" : "516fe13f0ac19c3ed7000001",
  "type" : "brochure"
}

PUT /resource

Request

{
	"_id":"516fe13f0ac19c3ed7000001",
	"type":"book"	
}

Parameters

  • _id
  • type <string> (optional)

Response

{
  "_id" : "516fe13f0ac19c3ed7000001",
  "type" : "book"
}
